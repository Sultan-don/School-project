# Подробная инструкция по запуску школьного сайта

Этот документ поможет вам шаг за шагом запустить проект на вашем компьютере.

## Шаг 0: Подготовка (Что нужно установить)

Перед началом убедитесь, что у вас установлены:
1.  **Node.js**: Скачайте и установите версию LTS с официального сайта [nodejs.org](https://nodejs.org/).
    *   Чтобы проверить, установлен ли он, откройте терминал (PowerShell или CMD) и напишите `node -v`. Должна появиться версия (например, `v20.11.0`).
2.  **Редактор кода**: Рекомендуется [VS Code](https://code.visualstudio.com/), но можно использовать любой.

---

## Шаг 1: Настройка Базы Данных (Supabase)

Весь контент сайта (книги, расписание, объявления) и пользователи хранятся в облаке Supabase. Настроим его:

1.  **Создайте проект**:
    *   Перейдите на [database.new](https://database.new) (или [supabase.com](https://supabase.com)).
    *   Зарегистрируйтесь/Войдите через GitHub.
    *   Нажмите "New Project".
    *   Придумайте имя (например, `SchoolSite`) и сложный пароль базы данных.
    *   Выберите регион (ближайший к вам, например, Frankfurt).
    *   Нажмите "Create new project" и подождите пару минут, пока он создается.

2.  **Создайте таблицы (структуру БД)**:
    *   В меню слева выберите иконку **SQL Editor** (выглядит как страница с кодом).
    *   Откройте файл `schema.sql` из папки этого проекта на компьютере.
    *   Скопируйте **весь** текст из файла `schema.sql`.
    *   Вставьте его в SQL Editor на сайте Supabase.
    *   Нажмите кнопку **Run** (зеленая кнопка справа внизу).
    *   Внизу должно появиться сообщение `Success`. Значит, таблицы созданы.

3.  **Получите ключи доступа**:
    *   В меню слева нажмите на шестеренку **Project Settings**.
    *   Выберите раздел **API**.
    *   Здесь вы увидите `Project URL` и `anon public key`. Не закрывайте эту страницу, они понадобятся на следующем шаге.

4.  **Создайте Администратора**:
    *   В меню слева выберите иконку **Authentication** -> **Users**.
    *   Нажмите "Add user" -> "Create new user".
    *   Введите Email (например, `admin@school.com`) и Пароль. Запомните их!
    *   Нажмите "Create user".
    *   *(Важно)*: По умолчанию Supabase требует подтверждения email. Чтобы зайти сразу без подтверждения:
        *   Зайдите в **Authentication** -> **Providers** -> **Email**.
        *   Отключите галочку "Confirm email".
        *   Нажмите "Save".

---

## Шаг 2: Настройка Проекта на Компьютере

Теперь свяжем наш код с созданной базой данных.

1.  Откройте папку проекта в VS Code (или проводнике).
2.  Найдите файл `.env.example`.
3.  Создайте новый файл с именем `.env` (просто `.env`, без названия перед точкой).
4.  Скопируйте содержимое `.env.example` в `.env`.
5.  Замените значения на свои (из Шага 1.3):

Пример файла `.env`:
```env
VITE_SUPABASE_URL=https://ваши-цифры.supabase.co
VITE_SUPABASE_ANON_KEY=ваш_длинный_ключ_из_supabase
```

---

## Шаг 3: Установка и Запуск

> **ВАЖНО:** Копируйте команды без кавычек и дополнительных символов. Только текст команды.

1.  Откройте терминал в папке проекта (в VS Code: Terminal -> New Terminal).

2.  Установите библиотеки (нужно сделать один раз).
    Введите в терминал команду:
    `npm install`
    Затем нажмите Enter.

3.  Запустите сайт.
    Введите в терминал команду:
    `npm run dev`
    Затем нажмите Enter.

4.  Терминал напишет что-то вроде:
    `➜  Local:   http://localhost:5173/`
    Зажмите `Ctrl` и кликните по ссылке `http://localhost:5173`, или просто скопируйте её в браузер.

**Готово! Сайт работает.**

---

## Как пользоваться

### Публичный сайт
Доступен всем по адресу `http://localhost:5173`.
Здесь можно смотреть расписание, книги и объявления. Пока база пустая, там будет пусто.

### Панель Администратора (для наполнения сайта)
1.  Перейдите по адресу: `http://localhost:5173/admin`
2.  Вас перекинет на страницу входа.
3.  Введите Email и Пароль, которые вы создали в Supabase (Шаг 1.4).
4.  После входа вы увидите меню:
    *   **Книги**: Добавьте пару книг для теста.
    *   **Расписание**: Создайте расписание уроков.
    *   **Объявления**: Опубликуйте новости школы.
5.  Вернитесь на публичный сайт (`http://localhost:5173`), чтобы увидеть добавленный контент.

### Если что-то не работает
- Если терминал пишет ошибку "Имя 'bash' не распознано", значит вы скопировали лишние слова. Вводите только `npm install` или `npm run dev`.
- Если сайт не грузит данные, проверьте файл `.env` — правильно ли вы скопировали URL и Key. Ключ должен быть `anon`, а не `service_role`.
- Чтобы остановить сервер, в терминале нажмите `Ctrl + C`.

Удачи в использовании!
