## 💡 Как это работает:

### Home.jsx (Быстрая загрузка с batch-просмотрами)
```jsx
// Раньше: 10 объявлений = 10 отдельных await для просмотров
// Теперь: 1 batch-операция
await trackViewsBatch(announcementIds, userIP);
```

### ManageBooks.jsx (Пагинированные таблицы)
```jsx
// Раньше: SELECT * FROM books (может быть 1000+ записей)
// Теперь: SELECT * FROM books LIMIT 15 OFFSET 0
.range((page - 1) * 15, page * 15 - 1)
```

### Агрегированные запросы (Объединённые counts)
```jsx
// Раньше: 3 отдельных запроса
// const likes = await select('*').count()
// const views = await select('*').count()

// Теперь: 1 запрос с JOIN
.select('...announcement_likes(count), announcement_views(count)')
```
