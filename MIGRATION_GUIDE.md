# Инструкция по обновлению базы данных

## Что нового?
1. ✅ **Реальные просмотры** - счётчик уникальных просмотров для каждого объявления
2. ✅ **Лайки один раз** - пользователь может лайкнуть объявление только один раз
3. ✅ **Поддержка видео** - теперь можно загружать видео вместе с фото

## Шаги обновления

### 1. Обновить схему БД (ВАЖНО!)

Скопируйте содержимое файла `schema.sql` целиком и выполните его в Supabase SQL Editor:

1. Откройте Supabase Dashboard
2. Перейдите в SQL Editor
3. Скопируйте **весь текст** из `schema.sql`
4. Вставьте в SQL Editor и нажмите **Run**

### 2. Обновить существующие объявления (Опционально)

Если у вас уже есть объявления в БД, запустите эту команду в SQL Editor:

```sql
-- Обновить старые объявления, чтобы добавить новые поля
UPDATE public.announcements 
SET 
  images = COALESCE(images, '{}'),
  videos = COALESCE(videos, '{}'),
  views_count = COALESCE(views_count, 0)
WHERE images IS NULL OR videos IS NULL;
```

### 3. Готово!

Теперь все новые функции работают:
- 📊 **Просмотры** - автоматически считаются по IP
- 👍 **Лайки** - один лайк на объявление от пользователя
- 🎥 **Видео** - загружайте видео при создании объявлений

- **IP адреса используются для отслеживания**: просмотры и лайки привязаны к IP пользователя
