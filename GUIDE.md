# 🚀 Как запустить твой сайт в интернет (для детей)

Привет! Если ты читаешь это, значит сайт готов. Теперь пора показать его всему миру. Вот как это сделать за 4 простых шага.

---

### Шаг 1: Загрузи код на GitHub (Твое хранилище)
GitHub — это как "флешка" в интернете, где будет лежать твой код.
1. Зайди на [github.com](https://github.com) и создай аккаунт.
2. Нажми кнопку **"New"** (создать новый репозиторий).
3. Назови его, например, `school-website`.
4. Загрузи туда все файлы из папки `school project` (кроме папки `node_modules`).
   *Если умеешь пользоваться программой **GitHub Desktop**, это будет еще проще!*

---

### Шаг 2: Подключи сайт к Vercel (Твой бесплатный сервер)
Vercel — это "дом" для твоего сайта, который делает его доступным по ссылке.
1. Зайди на [vercel.com](https://vercel.com) и войди через свой GitHub-аккаунт.
2. Нажми кнопку **"Add New"** -> **"Project"**.
3. Ты увидишь свой список из GitHub. Нажми **"Import"** рядом с `school-website`.
4. **ОСТАНОВИСЬ!** Перед тем как нажать Deploy, перейди к Шагу 3.

---

### Шаг 3: Настрой "Секретные ключи" (Environment Variables)
Чтобы сайт мог общаться с базой данных (Supabase), ему нужны ключи.
1. В настройках Vercel (прямо перед деплоем) найди раздел **"Environment Variables"**.
2. Добавь два ключа (скопируй их из своего файла `.env`):
   - Имя: `VITE_SUPABASE_URL` | Значение: (твоя ссылка)
   - Имя: `VITE_SUPABASE_ANON_KEY` | Значение: (твой длинный ключ)
3. Теперь нажми кнопку **"Deploy"**. Жди 1 минуту... БА-БАХ! Твой сайт в сети! 🎊

---

### Шаг 4: Как подключить свой Домен (Например: my-school.uz)
Если ты купил красивое имя (домен), вот как его привязать:
1. В Vercel зайди в свой проект и нажми вкладку **"Settings"** (сверху).
2. Слева выбери **"Domains"**.
3. Введи свое купленное имя (например, `school37.uz`) и нажми **"Add"**.
4. Vercel покажет тебе настройки (обычно это строки типа `A record` или `CNAME`).
5. Тебе нужно зайти туда, где ты купил домен (например, reg.ru или uz.uz), и вставить эти настройки в раздел **"DNS"**.
6. Подожди от 1 до 24 часов, и сайт заработает по твоему адресу!
